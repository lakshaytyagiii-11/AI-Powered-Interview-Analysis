AI-Powered Interview Analysis

An AI-based web application that helps users improve their interview performance through mock interviews, speech analysis, and detailed feedback reports.

🚀 Features

* 🎙️ AI Mock Interview Practice
* 🗣️ Speech & Filler Word Analysis
* 📋 STAR Method Answer Evaluation
* 📊 Performance Dashboard & Charts
* 🎯 Role-Based Interview Questions
* 📉 Confidence & Communication Analysis
* 🧠 Transcript Upload & Analysis
* 📄 Downloadable Interview Reports
* 🔊 Text-to-Speech Support
* 🌐 Fully Client-Side Web Application
⸻

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Web Speech API
* Canvas Charts

📌 How It Works

1. Select a target job role
2. Start a mock interview or upload transcript
3. Answer interview questions
4. The system analyzes:
    * speech quality
    * confidence
    * answer structure
    * filler words
    * role relevance
5. Get detailed feedback and improvement suggestions
